# Yunze Prediction Webapp

这是一个可直接部署到 GitHub Pages 的静态网页应用。

入口文件：

```text
index.html
```

## GitHub Pages 发布

在 GitHub 创建 public repository 后，把本仓库推送到 `main` 分支，然后在：

```text
Settings -> Pages -> Build and deployment
```

选择：

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

保存后，公开网址通常是：

```text
https://你的用户名.github.io/仓库名/
```
